# PFSS
Power Facility Soh Streaming  (2023.03.31~)

## 개요
빅데이터 기술에 대한 이론을 공부하고 만약 전국에서 전력설비의 데이터를 실시간으로 받아서 soh 상태를 시각적으로 확인하면 손쉽게 이상 징후를 파악할 수 있지 않을까라는 생각에 시작하게 되었습니다.
